✨ 功能特色

  • Bangumi API 集成：直接搜索动画，查看详细信息
  • 多级评分系统：支持 1-10 分，包含半分级别
  • 拖拽排序：直观的拖拽操作，轻松调整排名
  • 自定义主题：20+ 渐变背景主题，支持自定义背景
  • 评论系统：为每部动画添加个人评论
  • 数据导出：导出高质量图片和 JSON 数据
  • Tag 分析：分析你的动画喜好标签
  • 响应式设计：支持桌面和移动端
